# aerthos

## components:

  * linux kernel
  * busybox
  * debian-based (deboostrap needed for iso creation)

## desktop-mode:

  * xinit
  * wmaker

## boot with:

  * cd/usb/hd
  * grub2 for now
  * qemu line
  * docker image otw


## should work with:
  * lvm
  * wifi
  * crappy laptop

# to create ISO:

  * git clone this repo
  * ```./make.bash all```
  * ```file *.iso```
  * ```ls -all *.iso```
